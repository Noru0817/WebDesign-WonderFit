회원가입 방법

01/08 2024

카테고리：회원가입

1. 하단의 'SNS 로그인' 중 원하는 미디어 클릭

2. 필수 약관 및 선택 약관 동의 후 '확인' 버튼 클릭

3. 본인이 사용할 닉네임 작성 후 '가입 완료' 버튼 클릭

스타일 검사 방법

01/10 2024

카테고리：스타일 검사

1. 각 카테고리 별 본인이 원하는 옷 이미지 선택

2. 모두 선택 됐을 시, '선택 완료' 버튼 클릭

3. 완료 이후, 본인 만의 스타일 유형이 계정에 저장

검색 방법

01/11 2024

카테고리：옷 검색

1. 메인 화면 우측 상단의 검색 박스 클릭

2. 검색 박스에서 제시해주는 분위기 키워드 클릭

3. 해당 분위기 조합에 맞는 옷을 입은 모델 사진 및 본인 검사 결과와 맞는 옷 제시

4. 제시된 내용 중 택 1
